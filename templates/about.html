What were we thinking?

As developers, programmers, and software engineers, we strive to not only complete our programming assignments and designs, we want to be able to lead, manage, and receive feedback in order to make the product development process manageable.

Where did it start?

Undergraduate studies in Computer Science, Software Engineering, Computer Engineering, and Electrical Engineering, assign a multitude of programming assignments to students. So along with version control applications i.e. GitHub, Bitbucket, etc., we wanted to provide a way to organize your assignments.

What makes this application different?

The software engineering and development fields have different methods for keeping up with assignments, and we want students in undergraduate courses to experience the same type of organization that is expected in real world software development.
